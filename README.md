# Bimetallic Nickel Manganese Phosphate (NiMnPO) Structures

## Overview
This repository contains the crystallographic information files (`.cif`) and VASP structural files (`.vasp` / `POSCAR`) for  surfaces used in the paper entitled **with various Ni:Mn substitution ratios.

## Structural Models
The following optimized structural models are included in this repository, categorized by their Ni:Mn atomic ratios:

*   **1:0** - Pure Nickel Phosphate (Baseline / No Mn substitution)
*   **11:1** - Ni-Mn Phosphate (11:1 ratio)
*   **5:1** - Ni-Mn Phosphate (5:1 ratio)
*   **3:1** - Ni-Mn Phosphate (3:1 ratio)

## File Formats
*   `*.cif`: Crystallographic Information Framework files. These can be visualized using standard crystallographic software such as VESTA or Mercury.
*   `*.vasp`: Structural files containing lattice vectors and atomic coordinates, ready to be used as input for VASP or easily converted for Quantum ESPRESSO.
